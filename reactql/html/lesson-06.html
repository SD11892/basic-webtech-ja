Lesson 06: メッセージ一覧

メッセージ一覧

メッセージ一覧を表示するようにする。

queryを変更

const query = gql`
  query {
    allMessages {
      text
    }
  }
`;

コンポーネントで表示

        <ul>
          { 
            (data.allMessages || []).map(({ text }) => (
              <li>{text}</li>
            ))
          }
        </ul>
