Lesson 08: eject

eject

ejectすると通常のReact Nativeアプリになります。

$ npm run eject
  • 開発環境が必要になります
  • ejectは非可逆です、要バックアッップ
  • Expo APIを使っているとExpoKitを使った構成になります
