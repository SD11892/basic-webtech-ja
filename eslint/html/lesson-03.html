Lesson 03: インストール

インストール

おすすめはローカルインストールです。 バージョンによって動きが変わることがあるためです。

例えば、以下のようにします。

mkdir sample-app
cd sample-app
npm init
npm install eslint --save-dev

課題

  • 新しいプロジェクトか既存のプロジェクトにeslintをインストールする

参考情報

  • https://www.npmjs.com/package/eslint
